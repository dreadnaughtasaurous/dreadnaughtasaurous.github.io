# How to Search

This wiki uses **Pagefind** ‚Äî a fast, client-side search engine that indexes every clause from every Enterprise Agreement. There are no external servers; all searching happens locally in your browser.

## Opening Search

Click the **Search** bar in the top navigation, or use a keyboard shortcut:

| Action | Shortcut |
|---|---|
| Open search | `Ctrl` + `K` (Windows) or `/` |
| Navigate results | `‚Üë` `‚Üì` arrow keys |
| Open a result | `Enter` |
| Close search | `Esc` |

## Basic Keyword Search

Type any word or phrase into the search box. Results are ranked by relevance and each result shows:

- The **clause title** and which EBA it belongs to
- A **highlighted excerpt** showing exactly where your keyword appears in the text
- Any **topic tags** attached to that clause

Every clause page carries plain-language synonyms alongside its legal terminology, so you can search using everyday language and still find the right clause.

### Using Both Filters Together

You can combine a keyword with both filters at once. The search uses **AND logic** ‚Äî a result must match the keyword *and* both selected filters to appear.

::: tip Example
To find overtime rules specific to Nurses and Midwives, type `overtime` in the search box and select **Nurses and Midwives 2024‚Äì2028** in the EBA dropdown. You can further narrow this by also selecting `overtime` or `penalty-rates` in the Topics dropdown.
:::

To reset filters, click the **Clear filters** button that appears when a filter is active.

## Tips for Better Results

- **Fewer words work better.** Search for `annual leave cashing out` rather than `can I cash out my annual leave entitlement`.
- **Use the topic filter first** if you know the subject area ‚Äî it dramatically narrows the result set before you even type.
- **Try synonyms.** If a search returns no results, try an alternate term (e.g. `redundancy` instead of `retrenchment`, or `on-call` instead of `recall`).
- **Results are capped at 25.** If you don't see what you're looking for in the first 25 results, add a filter or refine your keyword.

## Archive Search

The default search **does not include** superseded or expired Enterprise Agreements. If you need to look up historical entitlements ‚Äî for example, to interpret something that occurred under a previous agreement ‚Äî use the dedicated [Archive Search](/eba/archive/search/).

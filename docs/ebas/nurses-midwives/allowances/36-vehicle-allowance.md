# 36. Vehicle Allowance



**36.1** Employees required to use their own car, motorcycle or bicycle in connection with their duties will be paid the applicable vehicle allowance as set out in [Appendix 2](/ebas/nurses-midwives/appendices/02-wages-and-allowances). The minimum payment for each occasion of use is set out in [Appendix 2](/ebas/nurses-midwives/appendices/02-wages-and-allowances).

**36.2** In [Appendix 2](/ebas/nurses-midwives/appendices/02-wages-and-allowances), Cylinder refers to the number of cylinders in a vehicle's engine, and Electric car refers to a vehicle which has an electric motor instead of an internal combustion engine.

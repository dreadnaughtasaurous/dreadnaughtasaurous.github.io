# 51. Rest Breaks

**51.1** Employees will be entitled to a 10 minute rest break in each four hours worked, or part thereof being greater than one hour.

**51.2** Rest breaks will be taken at a time suitable to the Employer and will be counted as time worked.

Plain language terms: tea break, rest breaks.

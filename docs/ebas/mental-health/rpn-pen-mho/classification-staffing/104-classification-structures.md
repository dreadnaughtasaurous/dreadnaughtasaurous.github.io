# 104. Classification Structures

**104.1 Classification of positions**

- **(a)** The Employer shall ensure that the work of each Employee working in psychiatric services is classified in accordance with the relevant classification standards for the classification of:
    - **(i)** Registered Psychiatric Nurse
    - **(ii)** Psychiatric Enrolled Nurse; or
    - **(iii)** Mental Health Officer.
